# sistema-soporte-tecnico
Aplicación web para la gestión de soporte técnico e incidentes tecnológicos desarrollada con ReactJS y tecnologías web modernas.

## Descripción
Aplicación web orientada a la gestión de soporte técnico e incidentes tecnológicos dentro de una organización. El sistema permite registrar solicitudes, administrar tickets de soporte y realizar seguimiento a incidentes tecnológicos mediante una interfaz web moderna.

## Objetivo
Desarrollar una plataforma web que optimice el proceso de atención de soporte técnico utilizando tecnologías Front-End y Back-End.

## Tecnologías utilizadas
- ReactJS
- Axios
- API REST
- Context API
- Hooks
- JavaScript
- GitHub

## Funcionalidades principales
- Registro de incidentes tecnológicos
- Gestión de tickets de soporte
- Inicio de sesión de usuarios
- Seguimiento de solicitudes
- Navegación entre módulos
- Comunicación con API REST
